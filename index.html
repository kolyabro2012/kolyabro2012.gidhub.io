Есть три основных языка программирования:html, css, jawascript
html это структура, то есть по какому принципу будет строиться сайт. сss: каскадные стили страницы. Это цвет, то-есть разукраска страницы. jawascript это интерактивность, то-есть взаимодействие с пользователем(пример: я нажал на пуск а компьютер начал игру) Вместе их можно представить как тело: html это кости, css это кожа и одежда, а jawascript это вены, суставы и т.д.
